## 🚀 如何使用

### 1. 在你的电脑上克隆仓库

```bash
# 克隆仓库（需要先在GitHub/GitLab等平台创建仓库）
git clone <你的仓库地址> kk-website
cd kk-website
```

### 2. 安装依赖并启动

```bash
# 安装依赖
npm install

# 启动服务器（推荐）
./run.sh

# 或者使用npm
npm start
```

### 3. 访问网站

- **本地访问**：http://localhost:3000

### 4. 修改代码

```bash
# 修改前端
vim public/index.html

# 修改后端
vim server.js

# 提交修改
git add .
git commit -m "描述你的修改"
git push origin master
```

### 5. 更新到阿里云服务器

```bash
# 在服务器上
cd /path/to/kk-website
git pull origin master
./deploy.sh
```

## 📝 注意事项

### 1. 数据库文件
- `guestbook.db` 包含用户留言数据
- 需要妥善备份
- 不要提交到公共Git仓库（如果包含敏感数据）

### 2. 端口占用
- 默认端口：3000
- 确保端口未被其他程序占用
- 可以修改server.js中的PORT变量

### 3. 文件权限
- 确保服务器有读写权限
- 数据库文件需要可写权限
- 日志文件需要可写权限

### 4. 安全考虑
- 生产环境建议添加身份验证
- 使用HTTPS
- 定期备份数据
