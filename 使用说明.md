# kk哥的个人博客 - 使用说明

## 📋 项目概述

这是一个基于Node.js + Express + SQLite的个人博客网站，采用黑白配色，极简高级风格。

## 🚀 快速开始

### 1. 克隆仓库

```bash
# 在你的电脑上
git clone <你的仓库地址> kk-website
cd kk-website
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动服务器

```bash
# 方法1：使用npm
npm start

# 方法2：使用启动脚本
./run.sh

# 方法3：直接运行
node server.js
```

### 4. 访问网站

- **本地访问**：http://localhost:3000
- **公网访问**：http://101.37.105.81:3000

## 📁 项目结构

```
kk-website/
├── public/              # 前端文件
│   ├── index.html      # 主页面
│   ├── index.html.backup  # 备份文件
│   └── index.html.backup2 # 备份文件
├── server.js           # 服务器主文件
├── package.json        # 项目依赖
├── package-lock.json   # 依赖锁定文件
├── run.sh              # 启动脚本（推荐）
├── start.sh            # 启动脚本（备用）
├── guestbook.db        # SQLite数据库文件
├── .gitignore          # Git忽略文件
├── README.md           # 项目说明
└── 使用说明.md         # 本文件
```

## 🎯 功能说明

### 1. 博客展示
- 个人简介
- 兴趣爱好
- 损友故事（需要解锁）

### 2. 留言功能
- 任何人可以留言
- 留言持久化存储在SQLite数据库
- 留言公开显示

### 3. 损友内容解锁
- 需要输入特定外号才能查看损友内容
- 支持的外号：莫总、死人猪/啊猪、全部

### 4. 导航功能
- 左侧边栏导航
- 独立板块显示
- 点击导航栏切换板块

## 🌐 部署到阿里云ECS

### 1. 安装Node.js

```bash
# 安装Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
npm --version
```

### 2. 安装PM2（进程管理器）

```bash
npm install -g pm2
```

### 3. 克隆仓库

```bash
cd /var/www
git clone <你的仓库地址> kk-website
cd kk-website
npm install
```

### 4. 启动服务

```bash
# 使用PM2启动
pm2 start server.js --name kk-website

# 保存PM2配置
pm2 save
pm2 startup

# 查看PM2状态
pm2 status
pm2 logs kk-website
```

### 5. 配置Nginx（可选）

```nginx
# 编辑Nginx配置
sudo vim /etc/nginx/sites-available/kk-website

# 添加以下内容
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}

# 启用配置
sudo ln -s /etc/nginx/sites-available/kk-website /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

## 🔍 故障排除

### 1. 端口被占用

```bash
# 查看3000端口占用情况
lsof -i :3000

# 或者
netstat -tulpn | grep :3000

# 杀掉占用端口的进程
kill -9 <PID>
```

### 2. 数据库连接失败

```bash
# 检查数据库文件权限
ls -la guestbook.db

# 修复权限
chmod 644 guestbook.db

# 重新启动服务器
```

### 3. 依赖安装失败

```bash
# 删除node_modules和package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### 4. 服务器启动失败

```bash
# 查看详细错误信息
node server.js

# 或者查看日志
pm2 logs kk-website
```
